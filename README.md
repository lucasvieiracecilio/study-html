# Projeto de Estudo

Documento de apoio e referência para o projeto de estudo contido neste repositório.

## Descrição

Projeto simples para estudo e experimentação com HTML e recursos estáticos.

## Estrutura do projeto

- index.html — página principal do projeto
- assets/ — pasta para imagens, estilos e outros recursos estáticos

## Tecnologias

- HTML
- (opcional) CSS/JS dentro de `assets/`

## Como executar

1. Abra o arquivo [index.html](index.html) em um navegador.
2. Alternativamente, rode um servidor local (recomendado) e acesse `http://localhost:PORT`.

Exemplo com Python 3 (na pasta do projeto):

```bash
python -m http.server 8000
```

## Uso

Este repositório serve como base para criar páginas estáticas. Modifique `index.html` e coloque recursos em `assets/`.
